# MySQL Simulation Schemas

## Overview

This directory contains real-world database schema designs simulating various applications and platforms. These schemas demonstrate practical database design patterns used in production systems.

## Schemas

### AWS User Management (`aws-user/`)

Schema for AWS-like user and account management system:
* User accounts and authentication
* IAM roles and permissions
* Resource access control

### Dropbox (`dropbox/`)

File storage and sharing platform schema:
* File metadata and storage
* Folder hierarchies
* Sharing and permissions
* Version control

### E-Commerce (`ecommerce/`)

Complete e-commerce platform schema:
* Products and inventory
* Orders and payments
* Shopping carts
* Customer management

### Messaging (`messaging/`)

Messaging and chat application schema:
* User conversations
* Message threads
* Real-time messaging
* Message delivery status

### RecruitPro (`recruitPro/`)

Recruitment and hiring platform schema:
* Job postings
* Candidate management
* Application tracking
* Interview scheduling

### Restaurant (`restaurant/`)

Restaurant management system schema:
* Menu management
* Order processing
* Table reservations
* Staff scheduling

## Usage

These schemas serve as:
* Reference designs for similar applications
* Learning materials for database design
* Starting points for new projects
* Examples of real-world patterns

Each schema includes:
* Complete table definitions
* Relationships and foreign keys
* Indexes for performance
* Sample data structures
