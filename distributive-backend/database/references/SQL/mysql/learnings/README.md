# MySQL Learnings

## Overview

This directory contains advanced MySQL learning materials, patterns, and techniques used in production environments.

## Directory Structure

### Core Concepts

* **ctes/**: Common Table Expressions - advanced recursive and non-recursive CTEs
* **denormalizations/**: Advanced denormalization patterns for performance optimization
* **functions/**: User-defined functions (basic and advanced)
* **innodb/**: InnoDB storage engine specifics and optimizations
* **joins/**: Complex join patterns and optimization
* **locking/**: Advanced locking strategies and deadlock prevention
* **monitoring/**: Database monitoring and observability
* **performaces/**: Performance optimization (indexing, query optimization)
* **query-optimization/**: Query optimization techniques
* **replication-ha/**: Replication and high availability setups
* **stored-procedures/**: Stored procedure patterns
* **subqueries/**: Subquery patterns and optimization
* **transactions/**: Transaction management and isolation levels
* **triggers/**: Trigger patterns (basic and advanced)
* **views/**: View patterns and materialized views
